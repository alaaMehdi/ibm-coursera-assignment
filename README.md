# ci-cd-final-project

## Project Overview
This project demonstrates a complete CI/CD pipeline using:
- GitHub Actions (CI)
- Tekton (CD)
- OpenShift Pipelines

## Features
- Linting with flake8 / eslint
- Unit testing with nose / jest
- Container build and deployment
- OpenShift pipeline automation
